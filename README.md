# Big Renderer

DirectX 12 toy renderer written in C++. Loosely based on the [directx graphics samples](https://github.com/microsoft/DirectX-Graphics-Samples).

## Features

- [X] OBJ mesh loading
- [X] Texture mapping
- [ ] Scene loading (glTF2.0)
- [ ] Lights
	- [ ] Sun based shadow mapping
	- [ ] Point lights
	- [ ] Emssive surfaces
- [ ] PBR pipeline
- [ ] Transparency
